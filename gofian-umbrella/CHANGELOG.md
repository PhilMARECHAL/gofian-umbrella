## [2.2.0] - 2026-02-23

### Added — 13 modifications implemented (mods #9 to #21)
- **#9 Welcome Experience**: Typewriter animation (25ms/char) for first message, companion style greeting ("Hey! Moi c'est UmbrellaBuilder..."), no chips on first message, custom placeholder "Tapez votre prenom..."
- **#10 Chat Layout**: Messages anchored at bottom (justify-content: flex-end) like WhatsApp/iMessage — no dead space
- **#11 Product Clarity**: Welcome message explicitly mentions "prompts experts pour ChatGPT, Claude et Gemini" — user understands in 3 seconds
- **#12 Language Lock**: ABSOLUTE rule — ALL user-facing content in user's language. Only final prompt in English. NEVER mix languages. New rule 18 in Section 09
- **#13 Inspiration API**: Dynamic welcome with Wikipedia "On This Day" + Nager.at holidays. 30 fallback examples. Endpoint /api/inspiration with 1h cache. Injected into first message metadata
- **#14 Flow Fix**: Power Preview (Step 3B2) IMMEDIATELY followed by iterative loop (Step 3C). Not a stopping point
- **#15 Real Emojis**: All gamification emojis are now real Unicode characters (🌱⚡🔥💎🏆🟩⬜), not [bracketed descriptions]
- **#16 Free Text UX**: Hint "ou tapez votre reponse..." under all chips. "Autre chose" chip focuses input field instead of sending. Dashed border style for catch-all chip
- **#17 Progress Bar Math**: Strict floor(score/10) rule. 10 green squares = ONLY at 100/100. Enforced in system prompt
- **#18 Bootstrap Instruction**: Client prompts start with auto-execution instruction: "MUST immediately start executing... Do NOT summarize... Just START"
- **#19 Raw Button Removed**: Simplified prompt card footer — only Copy (gold) + Email (green). toggleRawView function removed
- **#20 Email Formatting**: Structured email with section separators (========), clear headers, copyright footer. Improved readability in plain-text email clients
- **#21 Copyright**: "Generated by GOFIAN Umbrella BUILDER (c) Philippe MARECHAL" added to: app footer, landing footer, prompt card footer, email signature, client prompt footer

### Changed
- `system_prompt.py`: Complete rewrite — PROMPTFORGE OS v2.2 (450 lines, 9 sections, 22 global rules)
- `config.py`: Added INSPIRATION_CACHE_SECONDS, GNEWS_API_KEY, new feature flags
- `app.py`: Added /api/inspiration endpoint with Wikipedia + Holidays APIs, 30 fallback examples, 1h cache
- `static/index.html`: Design tokens v2.2, chat bottom-anchor, typewriter JS, chip-hint CSS, copyright footers

### Fixed
- **Language mixing bug**: Chips and Power Preview no longer display English when user chose local language
- **Emoji rendering bug**: [zap], [fire], [gem] etc. now render as real emojis
- **Progress bar inaccuracy**: 92/100 correctly shows 9/10 green squares, not 10/10
- **Dead space**: Chat messages now start near the input field, not stranded at top
- **Email readability**: Prompt in email now has section separators and structure

## [1.4.0] - 2026-02-21

### Added
- **Modular architecture**: Split monolithic app.py into 5 focused modules
  - `config.py` — Constants and feature flags
  - `system_prompt.py` — PROMPTFORGE OS (AI brain, 10 sections)
  - `api_client.py` — OpenAI wrapper with retry logic
  - `session_manager.py` — Sessions, cleanup daemon, rate limiting
  - `app.py` — Thin Flask orchestration layer
- **Dual Mode**: Create new prompts OR Improve existing ones
- **F1 Engine**: Tree of Thought synthesis with 5 internal experts
- **40+ language support**: Country detection + cultural adaptation
- **Security layer**: Anti-injection, rate limiting (30/60s/IP), XSS protection
- **Progressive typing indicator**: 4-phase animation during generation
- **Celebration system**: CSS confetti animation on prompt delivery
- **Progress bar**: Step tracking (Step 1/3, 2/3, etc.)
- **Star feedback**: 5-star rating system for generated prompts
- **Prompt counter**: Trophy counter with gamification
- **Markdown rendering**: Headers, bold, italic, lists, code blocks
- **Prompt card**: Special rendering for generated prompts with copy button
- **Offline detection**: Banner when connection lost
- **Design system**: Navy/teal/beige palette, DM Sans + Nunito fonts
- **Responsive breakpoints**: 375px → 1200px+ with 4 tiers
- **Quick reply chips**: Side-by-side layout with ripple effect

### Changed
- Complete rewrite from v1.3 single-file architecture
- Backend: Flask on Gunicorn (production-ready)
- Deployment: GitHub → Render auto-deploy pipeline
